PAT1011 A+B 和 C

题目

给定区间[−231,231][-2^{31},2^{31}][−231,231]内的 3 个整数AAA、BBB和CCC，请判断A+BA+BA+B是否大于CCC。

输入格式

输入第 1 行给出正整数 T(≤10)T(≤10)T(≤10)，是测试用例的个数。随后给出TTT组测试用例，每组占一行，顺序给出AAA、BBB和CCC。整数间以空格分隔。

输出格式

对每组测试用例，在一行中输出Case #X: true 如果A+B>CA+B>CA+B>C，否则输出Case #X: false，其中XXX是测试用例的编号（从 1 开始）。
